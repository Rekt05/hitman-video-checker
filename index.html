Runs with unavailable youtube videos

    This website was made for Hitman WoA's SRC mods to find and reject runs where their youtube videos have become unavailable, other video platforms are not checked but a list of them are stored in a json file

    The site will initiate the update every day at 10pm UTC but updates will probably only be visible some time after that, if you have rejected/edited a run it will still show on the site until the next auto update or you can dm me on discord to initiate a manual update

    The open 25 runs button will open 25 runs at once, if it doesnt you may need to enable pop ups or whatever else, if you would like to reset the counter, button or list, you can refresh the page
